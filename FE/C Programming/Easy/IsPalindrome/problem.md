# 🟢 IsPalindrome

**Recommended Time:** 5 - 10 min

**Topics:** Branching, Loops, User I/O, Functions, 1D Array, String

## Description
Write a function named `isPalindrome` that determines whether a given string is a palindrome. A palindrome is a sequence of characters that reads the same forward and backward.

Treat the input exactly as given (case-sensitive). For example, `"Racecar"` is *not* a palindrome unless the case matches exactly.

`isPalindrome` is expected to take in one string as its argument and return `1` for a palindrome or `0` otherwise.

## Input/Output
- Input: a string
- Output: `1` if the word passed in is a palindrome, `0` otherwise.

## Examples

```text
Input: "radar"
Output: 1
Explanation: radar spelled backwards is radar
```
```
Input: "hello"
Output: 0
```
```
Input: ""
Output: 1
```
```
Input: "a"
Output: 1
```
```
Input: "Racecar"
Output: 0
Explanation: Case sensitivity
```
```
Input: "racecar"
Output: 1
```
